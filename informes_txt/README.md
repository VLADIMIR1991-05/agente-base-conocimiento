# Informes TXT

Aqui se guardan automaticamente los informes diarios de consultas.

Formato del archivo diario:

```text
informe_YYYY-MM-DD.txt
```

Ejemplo:

```text
informe_2026-07-09.txt
```

Los archivos `.txt` reales no se suben al repositorio porque contienen preguntas, respuestas, usuarios y datos internos. En local se crean en esta misma carpeta cuando el asistente registra consultas.
