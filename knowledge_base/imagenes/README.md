# Imagenes

Sube aqui imagenes reales que quieras que el agente pueda mostrar:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

Usa nombres descriptivos, por ejemplo:

```text
bardolino_tapa_adhesiva_0520000947.jpg
grafito_texturizado.jpg
```

Si la imagen esta en internet, es mejor poner el enlace en `knowledge_base/links/imagenes.json`.
